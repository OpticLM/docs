---
title: What is an Agent
---

LLMs are inherently designed only for conversation and cannot integrate into your daily life or work.
However, through tools like API calls, we've given LLMs the ability to take action.
We refer to this as an “Agent.”
An Agent typically refers to an LLM within a specific application that can assist you with your work.

Optic is an Agent client. To learn about and use Optic's Agent capabilities, please continue reading.
